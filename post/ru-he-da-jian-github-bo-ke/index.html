如何搭建github博客

2020-02-25 2 min read

1.搭建博客的前期准备

1.1 注册Github账号

1.2 创建一个仓库： Create a repository

  • 设置仓库名称时，它的格式是固定的，采用用户名.github.io的格式。
    因为这个仓库的名称就是博客的域名（地址）
  • 勾选Initialize this repository with a README
  • 点击底部的Create repository就完成了仓库的创建

1.3 创建token

  • 点击右上方的账号头像，选择Settings
  • 在设置菜单中，按照Developer setting --> Personal access tokens的路径进入，点击右上方的Generate new token
  • 在New token设置页面，Note输入对Token的描述或备注，勾选下方的repo
  • 最后将页面拉至底部，点击Generate token,就会生成一串由数字和字母组成的字符，这就是刚刚生成的token

1.4 安装Git

1.5 安装博客写作客户端

Gridea

是一个静态的博客写作客户端，它提供了免费和收费的博客主题，通过应用不同的主题，可以让原本简陋的博客更有设计感。

Gridea配置

  • 填写好信息，先点击保存，接着点击检测远程连接，查看是否连接成功
